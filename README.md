# HXPhotoBrowser

* A picture browser that simulates progressive loading

* Based on SDWebImage

# Language

Objective-C

# CocoaPods



# Example

* fade in style

```
#import "HXUIButton+SDWebImage.h" or import "HXUIImageView+SDWebImage.h"

- (void)sd_setFadeImageWithURL:(nullable NSURL *)url;
```
